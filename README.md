# Python Quiz App

Bu proje, Python ile yazılmış basit ve etkileşimli bir quiz uygulamasıdır.  
Terminal üzerinden çalışan uygulama, soruları sırayla gösterir ve kullanıcıdan cevap alarak doğru-yanlış değerlendirmesi yapar.

---

## Özellikler
- Çoktan seçmeli sorular
- Doğru cevap kontrolü
- Anlık skor hesaplama
- Quiz sonunda toplam skor gösterimi
- Temiz ve anlaşılır sınıf (class) yapısı

---

## Kullanılan Teknolojiler
- Python 3
- Class (OOP) yapısı
- Terminal/CLI uygulaması
